Module Aquarium::Aspects::DefaultObjectsHandler
In: lib/aquarium/aspects/default_objects_handler.rb

Some classes and modules support a :default_objects flag and use it if no type or object is specified. For "convenience", requires that classes and modules including this module have a hash @specification defined with keys :default_objects, :types, and :objects.
