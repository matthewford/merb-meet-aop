Public Class methods

new(options)

Create a join point object. It must have one and only type or object and one method or the special keywords :all. Usage: 

 join_point = JoinPoint.new.find :type => ..., :method_name => ... [, (:class_method | :instance_method) => (true | false) ]

where

:type => type_or_type_name_or_regexp:A single type, type name or regular expression matching only one type. One and only one type or object is required. An error is raised otherwise.

:method_name => method_name_or_sym::

:method => method_name_or_sym:A single method name or symbol. Only one is allowed, although the special flag :all is allowed.
(:class_method | :instance_method) => (true | false):Is the method a class or instance method? Defaults to :instance_method => true.
