make_set(*value_or_set_or_array)

Return a set containing the input item or list of items. If the input is a set or an array, it is returned. In all cases, the constructed set is a flattened version of the input and any nil elements are removed by strip_nils. Note that this behavior effectively converts nil to +[]+.
